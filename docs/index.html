How TRM Works

  Input x ─────────────────────────────────────────┐
                                                   │
  ┌────────────────────────────────────────────────▼────┐
  │                    RECURSIVE LOOP                   │
  │  ┌────────────────────────────────────────────────┐ │
  │  │  for k = 1 to K improvement steps:             │ │
  │  │    ┌──────────────────────────────────────┐    │ │
  │  │    │  for n = 1 to N reasoning cycles:    │    │ │
  │  │    │    z = L_level(z, z_H + x)           │    │ │  ← Update latent
  │  │    └──────────────────────────────────────┘    │ │
  │  │    z_H = L_level(z_H, z)                       │ │  ← Update answer
  │  └────────────────────────────────────────────────┘ │
  └────────────────────────────────────────────────────┘
                           │
                           ▼
                    Output y = LM_head(z_H)
